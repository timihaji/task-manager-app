Project detail v3 — six approaches in the app's own style

Locking the functional pattern: permanent dense status bar(s) + tab-driven body (from the v2 Tabbed Command Bar). Now varying layout, density, and tone — but every mockup uses the app's actual design tokens: Geist font, dark teal background #071512, teal accent #0f766e, 2px corners, 9.5px UPPERCASE labels, mono numbers.

Common ingredients across all six: project crumb + ID, status pill, all KPIs always visible, tab strip (Board / List / Brief / Decisions / Activity / Files), kanban body. Differs only in where each piece sits and how loud it is.

01Toolbar Stack
3 horizontal barsmost app-nativedensity: medium

Tradeoff: Three thin stacked bars — crumb, metrics, tabs — exactly mirroring the existing .topbar + .col-hdr rhythm. Most familiar muscle memory; trades 96px of vertical space for dense always-on context.
